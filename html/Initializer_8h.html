Definition: Initializer.h:116

AGX_SHUTDOWN_CALLBACK is used to run code when agx::shutdown is called.

Note the difference to AGX_SHUTDOWN_CALLBACK, which is only called once when the agx library is unloaded, agx::shutdown on the other hand can be called many times. Example usage:

AGX_SHUTDOWN_CALLBACK() { printf("agx::shutdown called!\n"); }
